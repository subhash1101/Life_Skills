# Good Practices for Software Development

## Which points were new to you?

* The idea of deep work is new to me.
* Deep work states that "Professional activity performed in a state of distraction-free concentration pushes your cognitive capabilities to their limit."
* The above statement means that work done with full concentration can push one's capability to its limit.
* Through deep work, one can do work more efficiently.
